# Sourdough CSS forms

Form styles for Sourdough

## Installation

Install [nodejs](http://nodejs.org) (comes with npm).

```
npm install sourdough-forms
```

## Build

Install dependencies:
```
npm install
```

Run the build:
```
npm run build
```

Run the test build:
```
npm run buildTest
```

## Browser support

* Modern Browsers
* Internet Explorer 9+
