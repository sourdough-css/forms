# Sourdough CSS forms

Form styles for Sourdough

## Installation

Install [iojs](http://iojs.org) (comes with npm).

```
npm install sourdough-forms
```

## Build

```
npm install
```

To generate a build:

```
npm run build
```

To create the test page:

```
npm run buildTest
```

## Browser support

* Modern Browsers
* Internet Explorer 8+
